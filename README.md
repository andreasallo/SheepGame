# SheepGame
Modificaciones y Funcionalidades Añadidas

Sistema de Dificultad Progresiva:
Implementado en SheepSpawner.cs. El intervalo de aparición de las ovejas disminuye gradualmente a medida que avanza el tiempo, aumentando la intensidad y el reto del juego conforme el jugador progresa.

Feedback Visual Mejorado (Efecto De Aumento de oveja):
Al golpear a una oveja, he modificado el efecto de escalado (TweenScale) que hace que la oveja crezca antes de desaparecer, brindando una respuesta visual mucho más dinámica y satisfactoria. (Por eso el corazon ya no sale)

Audio Ambiental:
Implementación de un MusicManager con persistencia entre escenas (DontDestroyOnLoad). Esto permite que la música de fondo se mantenga continua al navegar entre la pantalla de título y la escena de juego sin cortes abruptos.

Personalización de la UI:
He rediseñado la paleta de colores de la pantalla de título.
